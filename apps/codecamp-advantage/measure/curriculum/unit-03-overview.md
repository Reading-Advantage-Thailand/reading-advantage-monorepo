# Unit 03 Overview: HTML & CSS Crash Course

**Phase:** A (Foundations)
**Periods:** 6
**Portfolio Project:** Personal Portfolio Website (structure and styling)

## Learning Objectives

By the end of this unit, the intern can:

1. Write semantic HTML5 (header, nav, main, section, article, footer)
2. Use heading hierarchy correctly (h1–h6)
3. Apply the CSS box model (margin, border, padding, content)
4. Create layouts with Flexbox (justify-content, align-items, flex-direction, gap)
5. Create layouts with CSS Grid (grid-template-columns, grid-template-rows, gap)
6. Build responsive designs with media queries and mobile-first approach
7. Style a complete multi-section portfolio page from scratch

## Technologies & Versions

| Technology | Version | Purpose |
|-----------|---------|---------|
| HTML5 | Living standard | Page structure |
| CSS3 | Living standard | Styling and layout |
| Chrome DevTools | Latest | Inspect and debug layouts |

## Portfolio Connection

The intern builds the full structure and styling of their Personal Portfolio Website:

- Hero section with name and tagline
- About section
- Skills section (grid layout)
- Projects section (card grid)
- Contact section
- Responsive navigation

## Key Concepts

- **Semantic HTML:** Use `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>` — not just `<div>`
- **Mobile-first:** Write mobile CSS first, add breakpoints for larger screens
- **Box model:** Everything is a rectangle; understand how margin, border, padding stack
- **Flexbox vs Grid:** Flexbox for 1D (row or column), Grid for 2D (rows and columns)
