# Unit 04 Overview: JavaScript Fundamentals

**Phase:** A (Foundations)
**Periods:** 8
**Portfolio Project:** Personal Portfolio Website (interactivity)

## Learning Objectives

By the end of this unit, the intern can:

1. Declare variables with `const`, `let` (never `var`)
2. Write functions (declarations, expressions, arrow functions)
3. Manipulate the DOM (select, create, modify, remove elements)
4. Add event listeners (click, submit, input, keydown)
5. Use arrays and objects (map, filter, find, destructuring, spread)
6. Write asynchronous code with Promises and async/await
7. Fetch data from an API using the Fetch API
8. Handle errors with try/catch

| Technology | Version | Purpose |

## Portfolio Connection

The intern adds interactivity to their Personal Portfolio Website:

- Smooth scroll navigation
- Contact form with validation
- Dark mode toggle
- Project data loaded from a local JSON "API"
- A dynamic skills section that filters by category

## Key Conventions

- **`const` by default**, `let` only when reassignment is needed, never `var`
- **Arrow functions** for callbacks and short functions
- **`async/await`** over `.then()` chains
- **Template literals** over string concatenation
- **Early returns** over nested if/else
