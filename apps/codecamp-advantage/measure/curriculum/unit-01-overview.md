# Unit 01 Overview: Dev Environment Setup

**Phase:** A (Foundations)
**Periods:** 2
**Portfolio Project:** Personal Portfolio Website (scaffold)

## Learning Objectives

By the end of this unit, the intern can:

1. Navigate the terminal confidently (cd, ls, pwd, mkdir, rm, cat, echo)
2. Install and configure VS Code with essential extensions
3. Install Node.js 20 (LTS) and verify the installation
4. Install pnpm 8.15.8 and understand why it's used over npm
5. Clone the codecamp-advantage monorepo and run `pnpm install`
6. Start the dev server and confirm the app loads in a browser

## Technologies & Versions

| Technology | Version | Purpose |
|-----------|---------|---------|
| VS Code | Latest stable | Code editor |
| Node.js | 20 | JavaScript runtime |
| pnpm | 8.15.8 | Package manager |
| Git | Latest stable | Version control (installed, used in Unit 02) |

## VS Code Extensions

- ESLint — JavaScript/TypeScript linting
- Prettier — Code formatting
- GitLens — Git integration
- Tailwind CSS IntelliSense — Tailwind class autocomplete
- TypeScript Error Translator — Human-readable TS errors

## Portfolio Connection

The intern creates the project folder that will become their Personal Portfolio Website. No code yet — just the workspace and tooling.

## Prerequisites

- A computer with admin/sudo access for software installation
- A GitHub account (created in Unit 02 if needed)
