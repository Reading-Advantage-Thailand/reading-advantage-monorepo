# Track: Visual Refresh: Define Unique Identity

## Overview
This track defines and implements a highly opinionated visual identity for the Advantage Games project. It aims to replace generic "AI slop" with a distinct, memorable aesthetic.

## Documents
- [Specification](./spec.md)
- [Implementation Plan](./plan.md)

## Outcome
Implemented the **Obsidian Grimoire** design system.
- High-contrast Monochrome (Black, White, Gold)
- Brutalist shapes (0px radius, heavy borders)
- Aggressive typography (Serif headers, Monospace body)
- Hard shadows and immediate hover states
