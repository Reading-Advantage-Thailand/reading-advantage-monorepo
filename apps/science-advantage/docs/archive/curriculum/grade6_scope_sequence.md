# Grade 6 Science Scope and Sequence

## Science Advantage App - Thai National Standards Alignment

**Total Instructional Time**: 80 hours (120 periods × 40 minutes)
**Periods per Week**: 3
**Total Weeks**: 40 weeks

---

## INSTRUCTIONAL MODEL

### Lesson Types

1. **Explicit Instruction (EI)**: Reading passage + embedded assessment
2. **Lab Investigation (LAB)**: Hands-on experiment + digital lab report
3. **Fun Review (FR)**: Engaging review activity to recharge
4. **Summative Assessment (SA)**: Comprehensive assessment

### Two-Week Cycle Pattern

- **Week A**: EI + EI + EI (3 reading lessons)
- **Week B**: EI + EI + LAB (2 reading lessons + 1 lab)
- **Every 4 weeks**: Replace one EI with FR, followed by SA

### Assessment Framework

**Formative (embedded after each lesson):**

- 10-15 multiple-choice questions (auto-graded)
- 5-6 short-answer questions requiring detailed explanations (AI-scored, teacher review)
- Real-time adaptive difficulty adjustment

**Summative (every 4-5 weeks):**

- 35-40 multiple-choice questions
- 10-12 short-answer questions
- Lab report analysis and practical application
- AI generates detailed analytics on conceptual understanding and skill development

**Lab Reports (digital template):**

- Background research and context
- Question/Problem with scientific rationale
- Hypothesis with theoretical basis
- Detailed materials and procedure
- Data collection with proper units, multiple trials, calculations
- Data analysis (graphs, statistics, patterns, relationships)
- Conclusion using claim-evidence-reasoning framework
- Sources of error and improvements
- AI flags: statistical errors, weak reasoning, missing error analysis, insufficient evidence
