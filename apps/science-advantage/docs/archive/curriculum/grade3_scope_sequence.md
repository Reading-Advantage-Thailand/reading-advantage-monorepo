# Grade 3 Science Scope and Sequence

## Science Advantage App - Thai National Standards Alignment

**Total Instructional Time**: 80 hours (120 periods × 40 minutes)
**Periods per Week**: 3
**Total Weeks**: 40 weeks

---

## INSTRUCTIONAL MODEL

### Lesson Types

1. **Explicit Instruction (EI)**: Reading passage + embedded assessment (most lessons)
2. **Lab Investigation (LAB)**: Hands-on experiment + digital lab report (every 2 weeks)
3. **Fun Review (FR)**: Engaging review activity to recharge (every 4 weeks)
4. **Summative Assessment (SA)**: Comprehensive assessment (every 4-5 weeks)

### Two-Week Cycle Pattern

- **Week A**: EI + EI + EI (3 reading lessons)
- **Week B**: EI + EI + LAB (2 reading lessons + 1 lab)
- **Every 4 weeks**: Replace one EI with FR, followed by SA

### Assessment Framework

**Formative (embedded in app after each lesson):**

- 5-8 multiple-choice questions (auto-graded)
- 2-3 short-answer questions (AI-scored, teacher review)
- Real-time progress tracking

**Summative (every 4-5 weeks):**

- 20-25 multiple-choice questions across strands covered
- 5-7 short-answer questions
- 1 lab report review component
- AI generates class report on common errors and misconceptions

**Lab Reports (digital template):**

- Question/Problem
- Hypothesis/Prediction
- Materials
- Procedure (often pre-filled)
- Data Collection (structured forms)
- Analysis
- Conclusion
- AI flags: missing data, incorrect units, incomplete reasoning
- Teacher dashboard: class-wide patterns and issues

**Period 7 (EI): Life Processes - Growth**
